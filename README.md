# 📄 SOW Analyzer (GenAI - Open Source)

This is a lightweight GenAI-based application that analyzes SOW (Statement of Work) documents and flags potential issues using a local LLM.

## 🚀 Features
- Upload SOW PDF
- Detect inconsistencies and risks
- Flag issues with severity
- Runs fully locally using open-source models (Ollama)

## 🧠 Tech Stack
- Python
- Streamlit
- pdfplumber
- Ollama (Mistral / LLaMA)
